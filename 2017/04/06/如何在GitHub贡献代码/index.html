如何在GitHub贡献代码

让我们来看一下如何在GitHub贡献代码

下载和安装SourceTree


这里有破解版

注册 GitHub 账号

fork

Fork 组织里的项目:克隆别人的代码库到自己的项目中，可以作为子模块的形式使用，或二次开发。
Fork 完毕之后，自己的项目仓库中就多了一个刚才 Fork 的项目。

clone

Clone 自己的项目到本地
使用 GUI 工具 SourceTree，Clone 自己项目仓库中的刚才Fork的项目。
Clone 之后，本地上就拥有了 Fork 的项目。

Commit

更新代码之后，
使用 Commit 操作来创建新的代码版本。
记得填写好commit的标题和描述

Push

push 操作的功能就是推送本地更新的代码版本到远端服务器/代码托管平台（如GitHub）中

Pull Request

回到 GitHub 上，点击 Pull Request 按钮，向代码原有者提出合并代码的请求，如果对方接受的你的请求，你对 Fork 项目的改动就会提交到原项目中，即向他人贡献了你自己的代码。
