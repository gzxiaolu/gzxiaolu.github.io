配置Github

1.创建代码库
点击页面右上角的加号，选择New repository

进入代码库创建页面：

在Repository name下填写yourname.github.io，Description (optional)下填写一些简单的描述（不写也没有关系），如图所示：

注意：yourname填写的是你github注册的名字

2.代码库设置
正确创建之后，你将会看到如下界面：

绑定域名

在阿里云买到域名后

首先找到控制台–>产品与服务–>域名–>域名列表
在你要设置的域名中点击解析->添加解析
记录类型CNAME->记录值输入yourname.github.io

然后在在Hexo目录里的source文件下添加一个名为CNAME的文件，注意这个文件是没有后缀的，千万不要设置成.txt文本文件，文件的内容就是域名，格式如：

然后在github下yourname.github.io文件开启，点击界面右侧的Settings，你将会打开这个库的setting页面，向下拖动，直到看见GitHub Pages，如图：

然后在Custom domain下输入你的域名。
这样，你的网站就基本建成了。
